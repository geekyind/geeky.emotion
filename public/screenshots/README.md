## How to Capture Screenshots

1. **Run the application locally:**
   ```powershell
   # Start backend
   cd backend
   python app/main.py
   
   # Start frontend (in new terminal)
   cd frontend
   npm run dev
   ```

2. **Navigate to http://localhost:3000**

3. **Capture screenshots:**
   - Use Windows Snipping Tool (Win + Shift + S)
   - Or browser dev tools screenshot feature
   - Save with exact filenames listed above

4. **Optimize images:**
   - Use PNG format for UI screenshots
   - Compress images to keep file sizes reasonable (<500KB each)
   - Ensure high quality and readability

## Tips for Great Screenshots

- Use a clean browser window (hide bookmarks bar)
- Ensure consistent viewport size across screenshots
- Use actual representative content (not lorem ipsum)
- Capture at different screen sizes for mobile view
- Consider using anonymized/blurred personal data
- Ensure good lighting and contrast in UI elements
- Show interactive states (hover, focus) where relevant
